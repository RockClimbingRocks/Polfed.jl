# Function Reference

This page documents the top-level functions and constructors in `src/`.

For each entry you get:
- input parameters,
- behavior,
- related structs/types to inspect next.

## Notation

- Names ending with `!` mutate one or more arguments in-place.
- Many functions have several methods; each row below summarizes the method family.

## Public Entry Points

| Function | Inputs | Behavior | Related Structs |
|---|---|---|---|
| `polfed(mat, x0, howmany, target; ...)` | Matrix `mat`, starting vector/matrix `x0`, number of pairs `howmany`, `target`, config keywords | Main POLFED solve path for explicit matrices. Optionally enables optimized mapping, builds plans, computes filtered operator, runs Lanczos, and returns `(vals, vecs)` or `(vals, vecs, report)` | `MappingConfig`, `TransformConfig`, `FactorizationConfig`, `DoSConfig`, `Report` |
| `polfed(f!, x0, howmany, target; ...)` | Operator callback `f!(Y, X)` instead of explicit matrix | Same workflow as matrix path, but uses user operator directly | `MappingPlan`, `TransformPlan` |
| `lanczos(mat, x0, howmany; ...)` / `lanczos(f!, x0, howmany; ...)` | Matrix or linear operator, basis seed(s), convergence options | Standalone Lanczos/block-Lanczos eigensolver used by POLFED | `ConvergenceInfo`, `LanczosIterator`, `FactorizationReport` |
| `display_report(report; ...)` | Combined run report | Prints spectral-transform, factorization, and benchmark sections | `Report`, `SpectralTransformReport`, `FactorizationReport`, `BenchmarkReport` |
| `display_factorization_report(report, use_colors; ...)` | Factorization diagnostics | Prints convergence and timing details for Lanczos stage | `FactorizationReport` |
| `display_spectral_report(report, use_colors)` | Spectral transform diagnostics | Prints target/bounds/order/matvec-count summary | `SpectralTransformReport` |
| `display_benchmark_report(report, use_colors; ...)` | Timing and memory diagnostics | Prints total runtime and stage breakdowns, optionally memory | `BenchmarkReport`, `TimingReport`, `MemoryReport` |
